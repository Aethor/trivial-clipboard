# trivial-clipboard

trivial-clipboard let access system clipboard.

## Installation

```
(ql:quickload :trivial-clipboard)
```

## Usage

get text from clipboard.

```
(trivial-clipboard:text)
```

put text to clipboard.

```
(trivial-clipboard:text "put text!")
```
